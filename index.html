¿Que es el desarrollo de software?

El Desarrollo de software es un proceso para crear software de computadora utilizando uno o mas lenguajes de programacion que brindan funcionalidad para abordar objetivos comerciales o personales particulares suele ser una iniciativa planificada con una serie de pasos y etapas

Por lo tanto una buena definicion es un proceso con un conjunto de actividades que crean productos de software informatico , incluido su diseño , desarrollo prueba e implementacion

En si es un conjunto de instrucciones o programas que le dicen a una computadora que hacer es independiente del hardware y hace que las computadorassean programables

Proceso o ciclo de vida del desarrollo de software

Una pregunta comun es ¿cuales son los pasos en el desarrollo de software?

Los pasos precisos variaran segun cual de los modelos de desarrollo haya sido adoptado por la organizacion pero el desarrollo de cuaquier plan de desarrollo generico sera el mismo paracada proceso de desarrollo de software Cada modelo y enfoque incluira las mismas actividades de alto nivel en su modelo de proceso de desarrollo

  • Especificacion
  • Validacion
  • Evolucion
  • Diseño e implementacion

Toda organizacion que desarrolla software debe tener una hoja de ruta o plan de desarrollo que establezca todos los pasos de desarrollo que deben llevar a cabo

Eso es cierto ya sea que este utilizando un flujo de trabajo de desarrollo de software en cascada tradicional o un proceso de desarrollo de software agil La mayoria si no todas las organizaciones desarrollaran software continuamente mediante el desarrollo iterativo , agregando mas y mas funcionalidades con el tiempo . Un plan de desarrollo de software agil entregara esto en pequeños incrementos frecuentes repitiendo los pasos involucrados en el ciclo de vida del desarrollo de software entregando la nueva funcionalidad a intervalos mucho mas largos , aveces hasta un año

¿Cuales son los diferentes tipos de desarrollo de software?

El software pertenece a Tres grupos principalessegun su uso y aplicacion aqui estan las categorias principales

Software del sistema

Tambien llamado sistema operativo o OS , el software del sistema es el programa que utiliza la computadora para traducir los comandos de entrada a un lenguaje mas legible por maquina .El sistema operativo controla los componentes del hardware de una computadora

Software de la aplicacion

Esta es la aplicacion que la mayoria de la gente utiliza para realizar tareas en su computadora y telefonos inteligentes los ejemplos mas comunes son , aplicaciones de procesamiento de texto , navegadores de internet, reproductores multimedia herramientas de edicion de fotos , antivirus e incluso productos de software como servicio

Lenguajes de programacion

Este es el lenguaje de programacion utilizado para crear software, solo lo utiizan los codificadores para crear programas los lenguajes de programacion incluyen , Java , C++, PHP, Python
